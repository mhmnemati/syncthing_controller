# Syncthing Controller

Syncthing Controller is a Golang application responsible for syncthing cluster controlling.

## Installation

Use the golang cli [go](https://go.dev/) to install dependencies.

```bash
go get .
```

## Usage

```python
# run application
go run .

# test application
go test .

# build application
go build .
```
